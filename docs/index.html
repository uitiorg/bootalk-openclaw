Slack에서 부톡봇에게 말을 걸어 업무를 처리하세요

부톡봇 · Slack DM 또는 채널 멘션으로 사용
사용 방법 — 부톡봇에게 DM을 보내거나, 채널에서 @부톡봇을 멘션한 후 자연어로 요청하면 됩니다.
정확한 명령어를 외울 필요 없이 평소 말하듯 요청하면 알아서 처리합니다.
